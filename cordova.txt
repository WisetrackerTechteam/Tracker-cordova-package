Tracker Cordova 플러그인

1. 플러그인 저장소
https://github.com/WisetrackerTechteam/Tracker-cordova-android-plugin

2. 플러그인 동작 방식
- 고객사의 cordova 안드로이드 프로젝트에서 와이즈트래커가 제공하는 플러그인 파일들을 삽입하여 cordova <-> Native SDK 간 데이터 연동
- cordova 환경에서 전달 받은 데이터를 플러그인 WiseTrackerBridge.java 통해 Native SDK 호출

3. 플러그인 제공 파일

1) plugin.xml
- cordova 안드로이드 프로젝트의 AndroidManifest.xml 파일에 포함될 사용자 정의 설정 내용

2) android

(1) plugin.gradle
플러그인에서 사용할 라이브러리 설정 파일 (Saas SDK 의존성)
(2) .java
플러그인 소스코드

3) www/WiseTrackerCordovaBridge.js
cordova에서 플러그인으로 연동되도록 하기 위한 스크립트 파일

4. 플러그인 추출 방법
플러그인 수정시 git remote push
